s3fs-fuse encryption and decryption - in this project I downloaded the source code from the github account of the s3fs-fuse authors. The original source code allowed for a user to upload and download documents to Amazon S3 storage when s3fs-fuse is mounted on a bucket on your local linux client. My main purpose was to modify the source code to encrypt any files that were to be uploaded into S3. Another purpose of the project was to auto decrypt any files that are downloaded from my S3 bucket. My encryption algorithm that is used in the project is Rc4.

Skills acquired from secure-s3fs

  • C++

  • Linux/Unix platform
